Projects

Elden Ring Speedrun Optimizer

In Spring 2023, I and a few friends created a C++ program that reads in data from the Elden Ring map and calculates an optimal speedrunning route using various modifications of graph traversal algorithms like Dijkstra's and Floyd-Warshall to find shortests paths while simultaneously accounting for all the intricacies of the video game and enforcing certain pathways (you can't fight Radahn without visiting Altus first!). In the end, the calculated times for our routes were within 5% of actual speedrun world records! Finally, using the path we calculated, we used Python and Matplotlib to visualize the route via a path drawn over the actual Elden Ring map. I've tried the route myself in-game and it works pretty well!
